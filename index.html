브금



HTML


깃허브에 업로드 해둔 html 공부 페이지
html은 고등학교 시절 웹 프로그래밍을 공부할 때 생활코딩 강좌를 이미 봤었기 때문에 쉽게 볼 수 있었다.
그래서 이번엔 우분투에 서버를 설치해서 내가 만든 웹 사이트를 작동 시켜봤다.
그러나 워드프레스 설치에는 이번에도 실패했다. 아마도 db 설치를 잘못한것 같다.






CSS


깃허브에 업로드 해둔 css 공부 페이지
CSS도 예전에 반응형 웹 사이트 만들기를 해볼 때 살짝 경험해 봤다.
강좌를 보며 어떻게 접근해야 하는지 알게 되었다.
예전에 학종 면접볼 때 class 하고 id의 차이점을 아냐는 질문이 들어왔었는데 정확히 대답하지 못했었다.
지금은 이제 잘 대답할 수 있을 것 같다.
또 내가 만드는 웹사이트들을 잘 꾸밀 수 있게 된것 같아 좋다.






JAVASCRIPT

자바스크립트는 이번 기회에 처음 시작하게 되어 좋았다.
예전에 웹사이트를 처음 만들어봐야겠다고 생각했을 때 react나 vue를 다루는 형들에게 조언을 얻은 적이 있다.
자바스크립트를 배워두면 좋다고 했었는데 마침 좋은 기회가 있어 기초부터 차근차근 강좌를 봤다.
앞으로 자바스크립트를 공부하면서 node.js도 공부해 보고 싶다.






후기

그동안 컴퓨터 분야중 웹페이지를 제작하는 쪽에 제일 관심이 있었다.
공부해야지 해야지 말만했던 컴퓨터 언어를 익히게 되어 보람있다.
아마도 개발자가 된다면 웹쪽 스택을 계속 쌓고 싶다.
어느 스택을 쌓을지 경험해 볼 수 있는 좋은 기회였던것 같다.
백엔드에 해당하는 부분도 더 공부해 보고 싶다는 목표도 세우게 되었다.
